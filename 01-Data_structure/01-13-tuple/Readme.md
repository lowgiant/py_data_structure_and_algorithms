# 파이썬 자료구조 

## 1. 자료구조
>   ### (13) 튜플(tuple)
        쉼표로 구분된 값으로 이루어진 불변 시퀀스 타입

        01) count(x)
        튜플 속의 x의 갯수를 확인

        02) index(x)
        튜플 속의 x의 위치를 반환

        03) unpacking
        변수를 할당하는 문장에서 왼쪽에 두개 이상의 변수를 사용해 한 변수 앞에 *연산자를 붙여 오른쪽 값들 중 할당되고 남은 값들은 *변수에 할당된다.

        04) named tuple
        튜플의 항목을 인덱스 위치 뿐만 아니라 이름도 참조가 가능
