# 파이썬 자료구조 

## 1. 자료구조
>   ### (8) 소수(prime)
        소수란 자신보다 작은 두개의 자연수를 곱해 만들 수 없는 1 보다 큰 자연수를 말한다.
        
        1) 브루트 방식
           무차별 대입 방법으로 찾는다.
           ex) a > m 이면, b < m
        
        2) 제곱근 방식
           제곱근은 제곱하여 그 수가 되는 수를 가리킨다.
           ex) a = m 이면, b = m 

        3) 페르마의 소정리 방식
           확률론적 테스트와 페르마 소정리 방식을 활용해 소수를 찾는다.
           ex) a < m 이면, b > m
